# Security Policy

## Reporting a Vulnerability

Please **DO NOT** open public GitHub issues for security vulnerabilities (e.g., SQL Injection, Token Leaks).

If you discover a security issue, please email me directly.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |
